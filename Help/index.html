*NIXANZ Snail Mail

User Guide

Before Using Snail Mail

Before using Snail Mail, you should have some addresses entered into your Mac OS X Address Book (application distributed with Mac OS X), otherwise you will only be able to manually enter addresses each time you want to print an envelope. Preferably, you should have one of the entries in the Address Book marked as "My Card" (choose "Show My Card" from Address Book's "Card" menu to see if it's set, or choose "This is My Card" from Address Book's "Card" menu to set it for the selected entry). Snail Mail, by default, uses your card in the Address Book for the return address.

Selecting a Return Address

The return address is automatically set to the first address of the "My Card" record in the Mac OS X Address Book, by default.

If you have more than one address entered in the Address Book's "My Card" record, you can choose your return address from the address selection popup menu (next to the "Return Address:" popup menu), which automatically contains all of your "My Card" addresses. This selection will be remembered next time you use Snail Mail.

It is also possible to enter a return address manually (see "Manual Return Address" below) or to have no return address printed (see "No Return Address" below). To change back to having your address book automatically kept up to date with the Address Book, choose "Address Book" from the "Return Address" popup menu.

Selecting a Delivery Address (ie, Addressing the Envelope)

Select a delivery address (to whom the envelope is to be sent) by clicking on an address in the list of addresses (which contains all of the addresses from your Address Book)
OR
Type part of the name of the delivery address you want in the "Search" field, and the list of names and addresses automatically updates to show those which match the search text you type.

Printing Single Envelopes

To print immediately (no dialogue) just click the "Print" button in the toolbar! Snail Mail will attempt to print 1 copy of the envelope to the printer that was last used in Snail Mail (using the same print settings last used in Snail Mail).
OR
Press COMMAND-"P" for a standard print dialogue to choose another printer, enter number of copies, or change other print settings.

Printing Envelopes from the Address Book Application
(Using the Address Book Snail Mail Plug-In)

If the Snail Mail plug-in is installed, you can print envelopes from the Mac OS X Address Book application. Just click on the label pop-up menu next to an address in the Address Book application, and choose the "Print Snail Mail Envelope" command (the menu is usually labelled, "home" or "work", etc). This will automatically launch Snail Mail, address the envelope to the address you chose, and choose Snail Mail's print command.

To Install the Plug-In

Launch Snail Mail, and choose "Install Address Book Plug-In" from the "File" menu.

Refining the List of Displayed Addresses

Click on a group from the Groups list to display only the addresses from that group as defined in the Address Book (to add, delete or modify groups, use the Address Book application).

Enter text in the "Search" field to have the list of addresses automatically updated to show those who's names match the search text you type.

Printing Batches of Envelopes

To print a batch of envelopes select all of the persons or addresses to be printed (using SHIFT-click or COMMAND-click, or "Select All" from the "Edit" menu) and choose the "Print Selected Envelopes..." command from the "File" menu.

Depending on settings in preferences this will either print envelopes for all of the selected addresses (default), OR, print one envelope for each of the selected persons with an address matching the label specified in preferences (eg, "Home", "Work").

Manual Return Address

If you want to use a return address that is not automatically included from the address book, you can override the automatic entry of the Return Address by selecting "Manual" from the "Return Address:" popup menu. When this option is selected, the text of the Return Address on the envelopes will no longer be updated automatically from information in the Address Book database. Now click in the return address area in the envelope window, and enter the text (and/or images) for return address you want.

Turning on this setting also allows saving of images in the return address or no return address at all - see below.

Images in Return Address

Images can be pasted into the return address text on the envelope window. To have the images remembered between uses of Snail Mail, select "Manual" from the "Return Address:" popup menu. If more precise formatting of images is required, the entire return address can be saved as an image (using some other image processing application) and then copy/pasted or drag/dropped into Snail Mail's return address area on the Envelope window.

To include images without using manual return address, see "Background Text and Images" below.

No Return Address

If you do not want to have any return address on your envelopes, you can select "None" from the "Return Address" popup menu. The return address selection popup menu will then be disabled, but it's selection will still be respected for country hiding and barcode display.

Background Text and Images

The entire background of the envelope can have any arbitrary text added with the exception of the margins themselves (which are outside the printable area). To edit text in this area, simply click on the envelope window to bring it to the front, then click anywhere in the envelope window apart from the return address, recipient address, or margins areas (these areas are easier to see if you turn on margin guidelines in the format menu). Then type and edit your text in the normal way.

Graphics can also be added to this background area by copy/paste or drag/drop.

NB: Because this background area overlaps with the return address and recipient address areas, be careful not to add text or graphics that will interfere with the text in these areas. It may be necessary to press RETURN or TAB (or both) a few times to avoid typing visible text directly behind the return address, for example.

Tip: Using images in the envelope background allows company logos to be placed behind the return address area and be used in conjuction with an automatically placed address from the address book. Because the left margin of the return address and the left margin of the background area are usually the same an extra step is necessary if you wish to have a graphic placed to the left of the return address (as is often the case for a company logo, for example). To do this, open the "Text Attributes" window using the toolbar button, or the command in the "Format" menu, then select all the text in the Return Address Fields area, and choose "Show Ruler" from the "Format" menu. Using this ruler, drag the left line starting markers to the right enough to make room for the desired graphic, then click "Okay".

Envelope Profiles and Page Setup

An envelope profile is simply a set of standard "Page Setup" settings and Snail Mail "Margins" settings. Snail Mail is initially configured with one default envelope profile (#10 envelope with appropriate margins) called "Default".

Add, Delete or Rename Envelope Profile

Choose "Edit Profiles..." from the "Envelope Setup" poup menu in the Snail Mail window. This will display the "Envelope Profiles" window. From here you can click "New" to add a profile, "Delete" to delete the selected profile or double click a profile name to change it's name.

Note: You cannot delete the last profile left (there must always be at least 1 profile), and you cannot have 2 profiles with the same name.

Edit an Envelope Profiles Envelope Setup (Page Setup and Margins)

To change the settings for an Envelope Profile, choose the profile from the "Envelope Setup" popup menu, then use the "Page Setup..." command in the "File" menu to change paper size, orientation, etc, and use the "Margins..." command in the "Format" menu to change margins settings (see "Margins" below).

Margins

Use the "Margins..." command in the "Format" menu to change the margin settings for both the return address text area and the delivery address text area on the envelope.

There is also a "Margin Guidelines" command in the "Format" menu which toggles grey lines which can help you in seeing exactly where the margins are on the envelope window (these lines can be left on, if desired, without being printed).

Default Text Attributes

Snail Mail can remember the text attributes (eg, font, colour, size, style) that you prefer for both delivery address and return address. You can set different attributes for different fields (ie, The name can be larger and/or in a different font or colour to the rest of the address).

To set the default text attributes, click the "Text Attributes" button in the toolbar, or choose the "Address Text Attributes" command from the "Format" menu, then in the window that appears, select the field(s) for which you wish to change attributes, then use the standard "Font" panel (which is also on screen) to change the attributes for the selected field(s).

Inter-Application Features

Address Book Plug-In

Snail Mail gets it's address list directly from the Address Book database, however some people may prefer to be able to print envelopes directly from the Address Book application itself. Snail Mail makes this possible with it's Address Book plug-in. You can install the plug-in using the "Install Plug-In" command in the "File" menu (and remove it using the "Remove Plug-In" command). After installing, the Address Book program will need to be re-launched (if already running) in order to utilise the Snail Mail plug-in.

Once installed, the plug-in can be used by clicking on the roll-over menu (ie, the label, such as "work" or "home") next to any postal addrss in the Address Book application. Choosing the "Print Snail Mail Envelope" command from the menu that appears will cause Snail Mail to be launched and to print an envelope addressed to the chosen Address Book record.

System Services

The following 3 Snail Mail services are available to other applications. Please note that Snail Mail must be installed in a standard Applications location (eg, /Application) for these to work. Additionally, these services will not be available until after the next login after installation.
Selected Text to Snail Mail Envelope - Places the selected text from the current application into the delivery address text area in the Snail Mail envelope window.
Snail Mail Envelope Text to Current Document - Places the text from the Snail Mail Envelope window's delivery address text area into the current application's frontmost window, at the insertion point (replacing any selected text).
Selected Text to Snail Mail Print - Places the selected text from the current application into the delivery address text area in the Snail Mail envelope window, and invoke's Snail Mail's print command. Ie, address and print an envelope using any text from any Services Compliant application.

Other Inter-Application Features

Many Mac OS X Applications are not services compliant (ie, all Carbon applications, as of OS X 10.2.x). Therefore to supply similar functionality to these applications there is a "Copy Address" and "Paste Address" toolbar icon in Snail Mail's main window to copy/paste text quickly between the envelope window's delivery address text area and the system clipboard (where it can be copied/pasted into any other application). This is a little faster than selecting the text in the envelope window and manually copying and pasting text using the Edit menu.

Preferences

Using the "Preferences..." command in the "Snail Mail" menu, the behaviour of Snail Mail can be altered as described for each setting below. All of these settings are off by default.

Hiding Fields

Select the checkbox(es) for any "Hide..." field(s) to avoid having those fields displayed in addresses.

Enter text in the prefix or suffix field(s) to have that text printed before/after the address(es) on the envelope. For example, a useful return address prefix might be, "If undeliverable, return to:"; or a useful addresse prefix might be, "Attn: Accounting Dept".

Local Country Variants:

Some countries can be referred to by several different names (eg, "United Kingdom", "UK", "U.K."), so in order fro Snail Mail to be able to tell that these different names all refer to the same country as the return address, enter the names in this field (each on a separate line). Snail Mail will treat each of these as though it was the same as the return address country, for the purposes of country hiding and barcoding.

Return Address Popup Includes:

By default, the Return Address popup menu in the main Snail Mail window contains all addresses in the special "My Card" record in the Address Book (if it exists). If you want to be able to choose return addresses from some other Address Book record, choose either "All Addresses", to have every address listed in the Return Address popup, or choose a group (as defined in the Address Book) to show every address from that group.

Use Barcodes:

Choose a barcode scheme to have those type of barcodes displayed/printed on your envelopes, or choose "(None)" to stop using barcodes. NB: Barcodes will only be printed if Snail Mail thinks the return address and delivery address are for the same country. You can add other countries for which barcodes should always be printed in the "Local Country Variants" (see above).

Print Selected Envelopes

By default, the "Print Selected Envelopes" command ("File" menu) prints one envelope for each selected address, however, if some persons records in the Address Book have several addresses (eg, one for work and one for home), you may want to automatically choose home addresses only, or work addresses only.

To only print addresses that match a particular label (eg, "Home", "Work") enter the label into the "Only Print Label" field in the Preferences window. Using this setting, one envelope is printed for each selected person who has a matching address (the address actually selected in the main window is ignored, only the selected person is used).

If some records do not have an address that matches the label you've entered (eg, "Home", "Work"), you can select the "Print Other Address if No Matching Label" checkbox to print an envelope for another address (if the record has another address). This way, you can be certain to print an envelope for every person with an address, even if some of the person records do not have an address that matches the label you've entered.

Editting Delivery Addresses

You can make one-off changes by editting delivery address text directly in the envelope window, but such changes will not be saved in the Address Book.

To make permanent changes to an address, double-click the address in the address list, and the Address Book will open that record in edit mode. To add or remove addresses, click on Snail Mail's "Address Book" button to open the Address Book in browse mode. (see the Address Book documentation for more information on editting, adding and deleting addresses).
